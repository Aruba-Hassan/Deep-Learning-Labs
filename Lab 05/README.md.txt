# This is fifth lab of Deep learning.
We learn in this lab how to train Artificial Neural Network to predict the customer churn.
The following models we use;
• ReLU (Rectified Linear Unit): Activation function for hidden layers.
• Sigmoid function: Used in the output layer to produce probability between 0 and 1.
• Binary Cross-Entropy Loss: Measures the difference between predicted and actual values.
• Adam Optimizer: Improves learning efficiency.
